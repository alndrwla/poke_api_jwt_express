#Object: Define api pokemon

#Actions
- Login JWT
- Add pokemon
- Delete pokemon
- Search info of team
- Change pokemon

#REST Design:
- Add pokemon: POST /team/pokemons
- Search info of team: GET /team
- Delete pokemon: DELETE /team/pokemos/:id
- Change pokemon: PUT /team
